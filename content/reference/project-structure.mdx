# Project Structure

```text
app/                 Next.js app routes, layouts, and API routes
components/          dashboard UI components
components/ui/       reusable UI primitives
content/             Nextra documentation content
lib/                 shared helpers, state, auth, and types
public/              static assets
scripts/             build helpers (search index, theme patch)
```

## Important files

| File | Purpose |
| --- | --- |
| `app/page.tsx` | Main dashboard route. |
| `app/docs/[[...mdxPath]]/page.tsx` | Nextra MDX docs route. |
| `app/api/palworld/[...path]/route.ts` | Palworld REST proxy. |
| `lib/server-context.tsx` | Client-side dashboard state and polling. |
| `lib/panel-auth-store.ts` | File-backed panel credential store. |
| `lib/access-tier.ts` | Admin/mod password classification. |
| `lib/rate-limit.ts` | In-memory failed-login limiter. |
| `lib/demo.ts` | Demo-mode data. |
| `lib/fps-sampler.ts` | In-process FPS sampler, started from `instrumentation.ts`. |
| `lib/fps-ring.ts` | FPS history ring read/write helpers. |
| `docker-compose.yml` | Recommended deployment stack. |
| `Dockerfile` | Production image build. |
| `scripts/build-search-index.mjs` | Documentation search index builder. |
