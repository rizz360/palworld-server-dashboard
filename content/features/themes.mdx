# Themes

The dashboard includes multiple built-in visual themes. Operators can switch themes from the header without changing code.

## Theme behavior

Theme selection is a browser-side preference. It affects the local UI only and does not change server configuration.

## Styling stack

The interface uses:

- Tailwind CSS
- Radix UI primitives
- custom components in `components/` and `components/ui/`
- Sonner toast notifications

The goal is a lightweight custom UI rather than a large all-in-one component framework.
