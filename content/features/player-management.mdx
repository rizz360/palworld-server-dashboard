# Player Management

The roster shows online players and gives operators quick moderation controls.

## Player data

Depending on the Palworld API response, the dashboard normalizes player payloads into:

- display name
- account name
- player ID
- user ID
- IP address when available
- ping
- level
- world coordinates

## Search

The roster can be searched by player-identifying fields such as name or user ID.

## Actions

Admins can:

- kick players
- ban players
- unban players
- review banned-player state tracked by the dashboard

Moderators can use a smaller widget with the allowed moderation actions only.

## Safety notes

Moderation actions are live server operations. Confirm the selected player before kicking or banning, especially when multiple players have similar names.
