# Live Map

The live map visualizes player positions reported by the Palworld REST API.

## Features

- player markers
- grouped markers when players are close together
- zoom and pan controls
- optional fast travel markers
- optional boss tower markers
- dashboard/map tab switching

## Coordinate source

Player coordinates come from the normalized player payload returned by the server. If the Palworld API returns no position data, players may appear at a default or missing location.

## Privacy considerations

Live location data can be sensitive on active multiplayer servers. Limit dashboard access to trusted operators and use the moderator tier where full visibility is not appropriate.
